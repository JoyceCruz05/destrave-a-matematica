<h1>Destrave a Matemática</h1>

<p>Landing page para divulgação de aulas particulares de matemática (ensino
fundamental, médio e pré-vestibular), construída com HTML, CSS e
JavaScript.</p>

<h2>Funcionalidades</h2>

<P>Cabeçalho - Exibe o nome do projeto e navegação pelas pagínas (Inicio, Sobre, Metodo, Contato).<br>
Inicio - Apresenta a proposta inicial do projeto, que é ajudar os alunos em matemática.<br>
Sobre - Apresenta informações sobre o professor, formato das aulas e público-alvo.<br>
Método - Explica a metodologia em 3 passos (diagnóstico → explicação → prática). <br>
Contato - Disponiliza o Formulário de inscrição contendo (nome, e-mail, CEP, mensagem).<br>
Rodapé - Contém direitos autorais e finalização da página.</p>
